Use of alt in img

Why use alt in img?
Altough image's source and url is already mentioned in src attribute, we use alt in img tag because, this attribute designates text for the provided image in case the image cannot be displayed through some reason. Another reason may be to make the website friendly to specially abled person so that they would know which image is being loaded although they couldn't see. For example, if some AI has to spell out the image displayed, it can't but it can spell out the text written in alt attribute so s/he would know about the image too.
This attribue is for alternate text for a definit area, if in case an image wouln't be shown. This attribute gives us the alternative information for an image if in case a user for vivid purpose wouldn't be able to see it(of tortoise pased connection, wrong src attribute, or the user uses a screen reader.)
